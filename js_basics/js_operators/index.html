ARITHEMATIC OPERATORS

Addition (+) Adds two numbers.
Subtraction (-) Subtracts the second number from the first.
Multiplication (*) Multiplies two numbers
Division (/) Divides the first number by the second.
Modulus (%) Returns the remainder of division.
Exponent (**) Raises the first number to the power of the second.

Increment/Decrement OPERATORS

Post Increment (Increment++) first the actual value is returned then its value is increment.
Pre Increment (++Increment) Increment the value and returns it.
Post Decrement(Decrement--) first the actual value is returned then its value is decrement.
Pre Decrement(--Decrement) Decrement the value and returns it.
